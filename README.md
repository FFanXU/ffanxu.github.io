### 1. 下载仓库

```
$ git clone https://github.com/YunYang1994/YunYang1994.github.io.git
$ cd YunYang1994.github.io
$ git checkout source
```

### 2. 开始构建

```
$ bash init.sh
```

### 3. 开始编译

```
$ bash build.sh
```

### 4. 上传至 github 

```
$ cd build
$ hexo g
$ hexo d
```
